1.プロジェクトフォルダを作成
 mdir xxx

2.githubのリモートレポジトリを作成
githubの右上の+でnew repository作成
デフォルトのママcreate repository

プロジェクトフォルダ内で画面上に出てくるコマンドを実行
echo "# line2deepl" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:uteten/line2deepl.git
git push -u origin main


2.az functionをcreate
 デプロイタブでリモートレポジトリを選択すると、workflowがアップロードされる

3.workflowをローカルにpull
git pull origin main

4. vscodeでHTTP Triggerの雛形作成
vscodeでプロジェクトフォルダをオープン
 左側のAマークをクリックし、雷ボタン＞Create HTTP function＞Python(Programing Model V2)＞python3.10
 function_app.py
 host.json
 requirements.txt
 などができる

さらに＋雷マーク＞HTTP Trigger＞Append to function_app.py(Recommended)をクリック
api/function_app.pyにHTTP Triggerのコードが追加される

雲マークの上矢印を選択してアップロード
